Theory

This experiment focuses on performing Exploratory Data Analysis (EDA) using Python on two datasets: a custom Student Dataset and the Cars93 Dataset. The main objective is to develop an understanding of dataset creation, data manipulation, and basic statistical analysis using Python and the Pandas library. For the student dataset, we first created a DataFrame containing five students’ information including roll numbers, gender, department, and CGPA. Several operations were performed on this dataset, such as checking the shape to understand the number of rows and columns, accessing specific columns like department, retrieving information for a particular student (PRN 103), and performing statistical analysis including mean, median, and mode of the CGPA column. We also checked the data types of each column to understand how Pandas interprets numerical and textual data. Additionally, we modified the dataset by inserting a new column age for each student, deleting the gender column, and identifying the topper based on the highest CGPA. We also retrieved specific information such as the CGPA of PRN 105. These operations demonstrate fundamental data cleaning, filtering, and summary techniques which are essential in real-world data analysis.

For the Cars93 dataset, we imported the CSV file into a Pandas DataFrame and explored it to gain insights into real-world automobile data. The dataset contains 93 records and 10 columns including Manufacturer, Model, Type, Price, MPG.city, AirBags, Horsepower, Passengers, Rear.seat.room, and Luggage.room. Basic EDA operations were performed, such as checking the dataset shape, viewing the first and last few records using head() and tail(), and analyzing the structure and types of the data. This helped in understanding how data is organized and the types of information that can be extracted from a dataset. Through this experiment, we learned how to effectively use Pandas for reading data, inspecting columns, performing calculations, modifying the dataset, and summarizing important insights.

Overall, this experiment provides a comprehensive understanding of the fundamentals of EDA including data inspection, statistical analysis, data cleaning, and basic data exploration. It highlights the importance of understanding dataset structure, extracting meaningful information, and preparing data for further analysis. This foundational knowledge is essential for any data-driven application and serves as the first step toward more advanced data analysis, visualization, and machine learning tasks. By performing these operations, we gained practical experience in handling both small custom datasets and larger real-world datasets, reinforcing the importance of Python and Pandas in data analysis workflows.



Conclusion

This experiment provided hands-on experience in performing Exploratory Data Analysis using Python and Pandas. Overall, it highlights that EDA is a critical first step in any data analysis workflow and provides a strong foundation for more advanced data visualization, modeling, and machine learning tasks.
